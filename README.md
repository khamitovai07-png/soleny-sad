# Соленый сад

Статический сайт для бренда ферментированных овощей в маленьких стеклянных баночках.

## Что внутри

- `index.html` - главная страница.
- `styles.css` - все стили сайта.
- `assets/favicon.svg` - favicon для вкладки браузера.
- `site.webmanifest` - базовый manifest для публикации.
- `robots.txt` - разрешение индексации.
- `.nojekyll` - отключает Jekyll-обработку на GitHub Pages.
- `assets/images/fermented-jars-studio.png` - первый вариант изображения баночек.
- `assets/images/jars-luxury-studio.png` - премиальное hero-фото баночек.
- `assets/images/jars-serving-table.png` - фото подачи к столу.
- `assets/images/jars-gift-box.png` - фото подарочного набора.
- `assets/images/product-*.png` - отдельные фотографии вкусов для карточек.
- `docs/superpowers/specs/2026-05-16-fermented-vegetables-site-design.md` - дизайн-спек.

## Как открыть локально

Откройте `index.html` в браузере.

Если нужен локальный сервер:

```bash
python3 -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.

## Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите в него файлы проекта.
3. Откройте `Settings` -> `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`.
6. Сохраните настройки.

Через несколько минут GitHub Pages покажет публичную ссылку на сайт.

После публикации сайт будет доступен по адресу `https://khamitovai07-png.github.io/soleny-sad/`.

## Что поменять перед реальной продажей

- Уточнить цены.
- Уточнить стоимость доставки и условия бесплатной доставки.
- Заменить примерные отзывы на реальные отзывы покупателей.
- Заменить сгенерированные фотографии каждого вкуса на реальные, когда будет готова предметная съемка.
- Подключить форму заказа, корзину или ссылку на маркетплейс.
